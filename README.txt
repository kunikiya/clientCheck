clientCheck

ユーザーエージェントを基にクライアント端末の種類とキャリアを判別するシンプルなライブラリ。

作成者 kunikiya
WEBサイト http://kunikiya.jp/
動作確認バージョン PHP 5.3
作成日 2012/5/10

isMobile()
  クライアント端末がガラケーである場合にture、そうでない場合にfalse
isSp()
  クライアント端末がスマホである場合にture、そうでない場合にfalse
isTab()
  クライアント端末がタブレットである場合にture、そうでない場合にfalse
isPc()
  クライアント端末がそれ以外である場合にture、そうでない場合にfalse
isKddi()
  クライアント端末のキャリアがauである場合にture、そうでない場合にfalse
isDocomo()
  クライアント端末のキャリアがドコモである場合にture、そうでない場合にfalse
isSoftbank()
  クライアント端末のキャリアがソフトバンクである場合にture、そうでない場合にfalse
